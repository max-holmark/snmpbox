# SNMP BOX (Docker wrapper)

## How to start
- Run the following command:
```
sh ./install.sh
```

## How to use
- SNMP Scanner
```
snmpbox scanner
```
- SNMP Scanner help
```
snmpbox scanner --help
```

- SNMP Simulator
```
snmpbox simulator
```

- SNMP Simulator help
```
snmpbox simulator --help
```

The link to base library: https://github.com/inexio/snmpsim
